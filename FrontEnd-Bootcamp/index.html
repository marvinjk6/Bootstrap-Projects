Aprenda os Fundamentos

Para se tornar um desenvolvedor Front-end você precisa aprender 3 tecnologias que são a base para um desenvolvedor web.

HTML é o componente base da web, ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS é o responsável por dar estilo ao HTML, de forma resumida vai deixar o site mais bonito e amigável

JavaScript é uma linguagem de programação que vai trazer dinamismo para o site, através da manipulação dos elementos

React

O React JS é uma biblioteca JavaScript de código aberto que é utilizada para construir interfaces de usuário mais especificamente para aplicações de página única.

Por que usar React?

O React foi pensado para ser possivel a reutilização de códigos através de criação de componentes, quem já conhece html, css e js consegue aproveitar esses conhecimentos e implementar no React. Além de performance e testabilidade, as aplicações React são fáceis de testar
